# Student Management System

A simple Python-based Student Management System that allows users to add and view student records through a command-line interface.

## Features

* Add student records
* View all student records
* User-friendly menu-driven interface

## Technologies Used

* Python

## Project Structure

student-management-system/
├── main.py
├── README.md
└── requirements.txt

## How to Run

1. Install Python.
2. Download the project files.
3. Open terminal in the project folder.
4. Run:

python main.py
